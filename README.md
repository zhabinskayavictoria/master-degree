# [2. Машинное обучение с учителем](supervised-machine-learning)
### [2.1. Реализация линейной регрессии с градиентным спуском](./supervised-machine-learning/hw_1/)

Реализация линейной регрессии «с нуля» без использования готовых библиотек машинного обучения. Создан класс, который обновляет веса через градиентный спуск, минимизируя MSE. Модель обучена на датасете зависимости зарплаты от стажа; результат визуализирован линией регрессии.

`numpy, pandas, matplotlib.`

### [2.2. Реализация логистической регрессии с градиентным спуском](./supervised-machine-learning/hw_2.ipynb)

С нуля реализована бинарная логистическая регрессия: сигмоид-функция, log loss, градиентный спуск и предсказание с порогом. На сгенерированных данных проанализировано влияние скорости обучения и числа итераций на сходимость модели.

`numpy, matplotlib.`

### [2.3. Задача классификации на Kaggle с помощью логистической регрессии](./supervised-machine-learning/hw_3/)

Задача определения типа месторождения (суша/море). Выполнен полный EDA, обработка пропусков, балансировка классов (ADASYN), отбор признаков (Random Forest). Обучены и сравнены логистическая регрессия, дерево решений, KNN, Random Forest, XGBoost, LightGBM, CatBoost с подбором гиперпараметров. Лучший результат по F1 — у XGBoost.

`pandas, numpy, sklearn, imblearn, xgboost, lightgbm, catboost, matplotlib, seaborn.`

### [2.4. Задача регрессии на Kaggle с помощью линейной регрессии (light)](./supervised-machine-learning/hw_4/)

Прогнозирование критической температуры сверхпроводников. Проведены обработка пропусков, оценка важности признаков (Random Forest) и отбор 150 из 167 признаков. Сравнены линейная регрессия, дерево решений, Random Forest, градиентный бустинг, KNN, XGBoost, LightGBM, CatBoost. Лучший MSE показал XGBoost, на его основе сформирован финальный сабмит.

`pandas, numpy, sklearn, xgboost, lightgbm, catboost, matplotlib.`

### [2.5. Задача регрессии на Kaggle с помощью линейной регрессии](./supervised-machine-learning/hw_final/)

Предсказание стоимости занятий на основе текстовых и числовых данных. Включает предобработку, генерацию новых признаков, кодирование категориальных переменных LabelEncoder и OneHotEncoder, отбор признаков через mutual information и масштабирование. Обучены и сравнены различные модели регрессии: лучший результат у XGBoost с подобранными гиперпараметрами.

`pandas, numpy, sklearn, xgboost, lightgbm, catboost, matplotlib.`

# [4. Курсовая работа](coursework)

**Предподготовка и анализ данных о продажах автомобилей в Казахстане**

В рамках курсового проекта была решена реальная бизнес-задача автодилерского холдинга. Автоцентру «Меркур Авто» требовалось понять свое положение на рынке и выявить точки роста на основе данных о продажах автомобилей в Казахстане за девять месяцев 2019 года. Исходные данные находились в типичном для операционных систем «сыром» состоянии: содержали нестандартные форматы, неявные ошибки и лишнюю информацию, что делало невозможным проведение качественного анализа без тщательной предподготовки.

- Техническая цель: довести датасет до состояния, пригодного для анализа, и создать воспроизводимый скрипт очистки для автоматизации обработки подобных данных в будущем.

- Аналитическая цель: провести разведочный анализ (EDA) всего рынка и дать оценку позиции и ключевым показателям эффективности автоцентра «Меркур Авто».

`pandas, numpy, matplotlib, seaborn.`
